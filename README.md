## üì± Usage Guide

### Taking Photos
1. Click the **Camera** tab
2. Select desired filter from the filter buttons
3. Click the red capture button to take a photo
4. Photos are automatically saved to your gallery

### Managing Gallery
1. Click the **Gallery** tab
2. **Select Photos**: Click on photos to select them
3. **Bulk Actions**: Use "Select All" and "Delete Selected" buttons
4. **Individual Actions**: Hover over photos to see Edit/Download/Delete options

### Editing Photos
1. In the gallery, hover over a photo and click **Edit**
2. Use filter buttons for quick effects
3. Adjust brightness, contrast, and saturation with sliders
4. Click **Save Changes** to update the photo
5. Click **Download** to save the edited version

### Creating Photo Strips
1. Click the **Photo Strip** tab
2. Choose your preferred template (Vertical, Horizontal, Collage, or Fancy)
3. Select exactly 3 photos from the gallery below
4. Preview your strip in real-time
5. Click **Generate Strip** to create the final version
6. Click **Download Strip** to save it

## üö® Troubleshooting

### Camera Not Working
- Check browser permissions for camera access
- Ensure camera is not being used by another application
- Try refreshing the page

### Photos Not Saving
- Check browser storage permissions
- Clear browser cache if storage is full
- Ensure JavaScript is enabled

### Downloads Not Working
- Check browser download settings
- Ensure pop-up blocker is not blocking downloads
- Try different browser if issues persist
